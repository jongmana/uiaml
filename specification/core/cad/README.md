# ContentArea Diagram (CAD)

The ContentArea Diagram is a core diagram of U*ia*ML, that is used to model the content of a contentarea using the following contentarea elements (CAE).

* [Contentarea Element](cad-contentarea/README.md)
* [Image Element](cad-image/README.md)
* [Link Element](cad-link/README.md)
* [List Element](cad-list/README.md)
* [Text Input Element](cad-text-input/README.md)
* [Text Label Element](cad-text-label/README.md)
